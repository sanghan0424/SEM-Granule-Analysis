SEM 扫描电镜完整学习笔记

一、SEM 二次电子（SE）形貌测试原理

1. 二次电子核心定义

二次电子（Secondary Electron，SE）是入射电子束轰击样品表面时，样品中原子的核外价电子受到激发，获得足够能量后脱离原子成为的自由电子，是 SEM 形貌观察中最常用的信号。

其能量较低，一般小于 50eV，产生深度极浅（样品表面 0~10nm 范围内），对样品表面的微观形貌变化极其敏感。

2. 二次电子的产生机制

  1. 入射电子束与样品表面原子的价电子发生非弹性散射，价电子吸收散射能量，克服原子核的束缚力；
  2. 获得足够能量的价电子脱离原子，从样品表面逸出，形成二次电子；
  3. 若价电子吸收的能量不足，无法逸出样品表面，则仅发生原子的电离或激发，不产生二次电子信号。

3. 二次电子用于形貌观察的核心优势

  • 分辨率高：产生深度仅在样品表面 10nm 以内，常规 SEM 下二次电子像分辨率可达 1~3nm，能清晰观察纳米级微观形貌（如颗粒、晶界、台阶）；
  • 形貌衬度极强：二次电子产额与样品表面倾斜角度相关——凸起/棱边/尖角处更亮，凹陷/孔洞/背向处更暗，可还原三维形貌；
  • 成像视野清晰：信号收集效率高，信噪比好，无明显拖影，细节呈现完整。

4. 二次电子信号的收集与成像过程

  1. 信号产生：入射电子束在样品表面逐点扫描，激发产生二次电子；
  2. 信号收集：E-T 探测器加 +200~+500V 正高压，吸引低能二次电子并转化为电信号；
  3. 信号放大：电信号经前置放大器、主放大器逐级放大；
  4. 成像显示：放大后的电信号同步控制显示器亮度，扫描轨迹严格同步，还原样品形貌像。

5. 二次电子与背散射电子（BSE）核心区别

特征 二次电子（SE） 背散射电子（BSE）
能量 低，<50eV 高，接近入射电子能量
产生深度 浅，0~10nm 深，100~1000nm
核心作用 形貌分析，分辨率高 成分/衬度分析，分辨率低
产额影响因素 样品表面倾斜角度 样品原子序数（Z）

二、SEM 图像数据获取实操步骤

1. 从 Materials Project 获取锂电池正极 SEM 图

  1. 打开新版官网：https://next-gen.materialsproject.org/（无需注册）；
  2. 搜索框输入精准关键词：LiCoO2 SEM / NCM SEM / lithium battery cathode particle SEM；
  3. 左侧筛选「Media/Images」分类，仅显示带 SEM 图的结果；
  4. 选择带比例尺的颗粒图，右键「图片另存为」保存到本地。

2. 从 nbsdc.cn 获取金颗粒 SEM 图

  1. 访问链接：https://www.nbsdc.cn/general/dataSetHome#，完成免费注册/登录；
  2. 搜索「金颗粒 15nm SEM」「金颗粒 50nm SEM」，加入购物车；
  3. 打开购物车（菜单按钮 → 购物车 或 直接访问 https://www.nbsdc.cn/general/shoppingCart）；
  4. 勾选文件后点击下载，将图片整理到指定路径：data/raw_sem/metal；
  5. 重命名文件：SEM-金属颗粒-01.jpg（15nm）、SEM-金属颗粒-02.jpg（50nm）。

三、SEM 测试实操注意事项

  • 样品处理：非导电样品（陶瓷/高分子/粉体）需喷金/喷碳，避免电荷积累；样品表面需清洁无杂质；
  • 电压选择：纳米级精细形貌选 5~10kV 低电压，大尺寸宏观形貌选 20~30kV 高电压；
  • 工作距离：常规 5~15mm，距离越小分辨率越高；
  • 环境要求：常规测试在高真空环境，软质/易损样品可选用低真空/湿环境 SEM。

四、常见问题排查

  • 无法打开购物车：优先点击页面右上角蓝色菜单按钮，或直接访问购物车专属链接；
  • Materials Project 解析失败：换新版地址 https://next-gen.materialsproject.org/；
  • SEM 图无比例尺：下载前确认图片标注「scale bar」「μm/nm」，无比例尺的图不建议使用；
  • 文件路径错误：Windows 路径示例 D:\项目名\data\raw_sem\metal，macOS/Linux 路径示例 /Users/用户名/项目名/data/raw_sem/metal。
