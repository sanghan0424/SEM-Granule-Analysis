# Python for循环笔记（SEM颗粒分析场景）
## 核心用法
1. 遍历列表：for 元素 in 列表 → 遍历颗粒粒径列表
2. 遍历range()：for i in range(5) → 生成序号/批量处理
3. 遍历元组：for 元素 in 元组 → 固定业务阈值（如(0.5,5)）

## 颗粒分析常用示例
- 统计有效颗粒数：遍历粒径列表，计数符合条件的元素
- 批量处理SEM图像：遍历图像文件名，批量重命名/移动

# Python while循环笔记（SEM颗粒粒径输入场景）
## 核心语法
while 条件:
    循环体
    # 终止循环：break
    # 跳过本次：continue

## 核心场景
1. 无限循环（while True）：手动输入场景，直到输入指定字符退出
2. 条件循环（while count < 10）：计数循环，达到阈值退出
3. 异常处理：结合try-except，避免输入非数字导致程序崩溃

## 颗粒分析适配
- 无限循环接收粒径输入，输入"q"退出
- 异常处理：捕获非数字输入，提示用户重新输入
