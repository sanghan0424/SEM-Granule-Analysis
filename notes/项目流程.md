# SEM 颗粒分析项目流程
**更新时间**：2026-02-26
**核心目标**：基于 Python 实现 SEM 图像颗粒的自动检测、尺寸测量与数据可视化。

## 核心模块（执行顺序）
1. **图像预处理**：读取 SEM 图像 → 灰度化 → 二值化 → 降噪（去除干扰噪点）
2. **目标提取**：轮廓检测 → 筛选有效颗粒 → 颗粒数量统计
3. **尺寸测量**：像素尺寸计算 → 比例尺换算（像素→μm/nm）→ 异常值剔除
4. **数据统计**：数量/尺寸统计 → 尺寸分布分析 → 生成统计报表
5. **可视化**：尺寸分布直方图 → 颗粒标注图 → 可视化报告导出
